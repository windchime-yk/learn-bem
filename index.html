BEMについて

BEMとは

BEMとは、Block、Element、Modifierの略語です。Webサイトのコンポーネント化のためのフロントエンド設計方法のひとつで、厳格なclass名の命名ルールが特徴的な手法です。

BEMによるフロントエンドの設計 第1回 基本概念とルール - CodeGrid 
        
          .block__element_modifier {
            /* do anything */
          }

MindBEMdingとは

          
            .block__element--modifier {
              /* do anything */
            }
